# django-rest-framework

Ref.: https://www.django-rest-framework.org 

1. Install Latest Django Release
2. Start project: django-admin startproject setup .
3. Start App: python manage.py startapp escola
4. Run Migrations everytime there's an update in the class Models: python manage.py makemigrations --> python manage.py migrate
5. To execute the application run: python manage.py runserver
6. To access admin site, create the superuser first: python manage.py createsuperuser
